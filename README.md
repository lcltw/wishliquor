# The Whisky List - Responsive Design Clone

This project replicates the responsive design from https://thewhiskylist.com.au/

## Tech Stack
- Next.js 14 (App Router)
- Tailwind CSS
- Framer Motion for animations

## Getting Started

```bash
npm install
npm run dev
```

## Sections Implemented

1. **Top Banner** - Promotional TWL Rewards announcement
2. **Header** - Navigation with logo, search, account, cart
3. **Hero Carousel** - Featured brands with auto-scroll
4. **Community Section** - Benefits of joining
5. **Product Grid** - Horizontal scrolling whisky products
6. **Promotional Cards** - Loyalty program & buying guides
7. **Reviews Section** - Recent whisky reviews carousel
8. **News & Events** - Latest news and upcoming events
9. **Footer** - Links, newsletter, social media, payment methods

## Design Features

- Fully responsive (mobile-first)
- Dark theme with amber accents (#C9A227)
- Horizontal scroll carousels with navigation arrows
- Hover effects on cards and buttons
- Accessible navigation
